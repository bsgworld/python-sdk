# OtpMessageStatus

Message status. Possible values:delivered – the message was sent;accepted – message accepted for sending;undelivered – the message was not delivered.

## Enum

* `SENT` (value: `'sent'`)

* `ACCEPTED` (value: `'accepted'`)
