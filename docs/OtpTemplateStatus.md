# OtpTemplateStatus

Status of the template. After template creation, the status of the template will always be “requested.”

## Enum

* `REQUESTED` (value: `'requested'`)

* `APPROVED` (value: `'approved'`)

* `REJECTED` (value: `'rejected'`)

* `DELETED` (value: `'deleted'`)
